# Boundaries

Educational static analysis only. This tool uses local synthetic or user-supplied inputs, does not fetch live data, does not connect to brokers, does not place orders, and does not provide tax, legal, investment, buy, sell, or hold advice.

- No live market, account, broker, or banking data is fetched.
- No orders, allocations, optimizations, buy/sell/hold recommendations, or predictions are produced.
- Outputs are deterministic local review artifacts for records and professional discussion.
